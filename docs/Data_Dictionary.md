# SocialLens BI Data Dictionary

Warehouse schema: `social_dw`

## Dimensions

### `dim_time`
Time grain for posts and comments. `time_id` should be generated by ETL from the UTC timestamp, for example `YYYYMMDDHHMMSS`; reporting date/hour fields are derived in Vietnam time.

| Column | Type | Description |
| --- | --- | --- |
| `time_id` | bigint | Primary key for the source event timestamp. |
| `full_date` | date | Reporting date. |
| `full_timestamp` | timestamptz | Exact reporting timestamp. |
| `hour_of_day` | smallint | Hour from 0 to 23. |
| `day_of_week` | smallint | ISO weekday from 1 Monday to 7 Sunday. |
| `day_name` | varchar | Human-readable weekday. |
| `week_of_year` | smallint | ISO week number. |
| `month_of_year` | smallint | Month number. |
| `month_name` | varchar | Human-readable month. |
| `quarter_of_year` | smallint | Quarter number. |
| `calendar_year` | smallint | Calendar year. |
| `is_weekend` | boolean | True for Saturday and Sunday. |

### `dim_platform`
Social platform lookup. The MVP real extraction scope is YouTube-only; additional platform values are retained for future extension.

| Column | Type | Description |
| --- | --- | --- |
| `platform_id` | smallserial | Primary key. |
| `platform_name` | varchar | Unique normalized name. MVP data uses `youtube`. |
| `platform_type` | varchar | Platform category. |
| `api_source` | varchar | Upstream API or collector. |
| `is_active` | boolean | Whether the platform is currently loaded. |

### `dim_content_type`
Normalized content format lookup.

| Column | Type | Description |
| --- | --- | --- |
| `content_type_id` | smallserial | Primary key. |
| `type_name` | varchar | Content format such as `video`, `reel`, `livestream`, `image`, `story`. |
| `type_category` | varchar | Broader reporting category. |

### `dim_page`
Owned and competitor page lookup.

| Column | Type | Description |
| --- | --- | --- |
| `page_id` | bigserial | Primary key. |
| `platform_id` | smallint | References `dim_platform`. |
| `external_page_id` | varchar | Source system page/channel/profile id. |
| `page_name` | varchar | Display name. |
| `industry` | varchar | Business category for benchmarking. |
| `country_code` | char(2) | ISO-3166 country code when available. |
| `is_competitor` | boolean | True for competitor pages. |
| `follower_count` | bigint | Latest known follower/subscriber count. |

## Facts

### `fact_post`
Post-level performance fact table. Grain: one source post per platform.

| Column | Type | Description |
| --- | --- | --- |
| `post_id` | bigserial | Warehouse primary key. |
| `external_post_id` | varchar | Source post/video id. |
| `platform_id` | smallint | References `dim_platform`. |
| `time_id` | bigint | References publish/reporting time in `dim_time`. |
| `content_type_id` | smallint | References `dim_content_type`. |
| `page_id` | bigint | References `dim_page`. |
| `post_url` | text | Public URL when available. |
| `caption` | text | Post text/caption. |
| `hashtag_count` | integer | Number of extracted hashtags. |
| `reach` | bigint | Unique accounts reached. |
| `impressions` | bigint | Total views/impressions. |
| `likes` | bigint | Like/reaction count. |
| `comments` | bigint | Comment count. |
| `shares` | bigint | Share/repost count. |
| `saves` | bigint | Save/bookmark count. |
| `engagement_count` | bigint | Generated as likes + comments + shares + saves. |
| `engagement_rate` | numeric | Generated as engagement_count / reach * 100; NULL when reach is 0. |
| `virality_score` | numeric | Generated as shares / reach * 100; NULL when reach is 0. |

### `fact_sentiment`
Comment sentiment fact table. Grain: one analyzed comment.

| Column | Type | Description |
| --- | --- | --- |
| `sentiment_id` | bigserial | Warehouse primary key. |
| `external_comment_id` | varchar | Source comment id. |
| `post_id` | bigint | References `fact_post`. |
| `platform_id` | smallint | References `dim_platform`. |
| `time_id` | bigint | References comment/analyzed time in `dim_time`. |
| `sentiment_label` | varchar | `positive`, `neutral`, or `negative`. |
| `sentiment_score` | numeric | Score from -1.0 to 1.0. |
| `comment_text` | text | Optional comment text for audit/debug. |

## Analytical Views

| View | Purpose |
| --- | --- |
| `vw_post_performance` | Denormalized post performance for dashboard queries. |
| `vw_sentiment_daily` | Daily sentiment rollup by platform and page. |
| `vw_platform_content_summary` | Platform/content aggregate for executive cards. |
| `vw_best_posting_heatmap` | Day/hour posting heatmap metrics. |
